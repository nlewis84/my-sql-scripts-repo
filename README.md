# SQL Scripts Repository

This repository contains SQL scripts managed by Git for SQL.

## Structure

- `sql-scripts/staging/` - Scripts for staging database
- `sql-scripts/production/` - Scripts for production database

## Process

1. Create a branch
2. Add SQL script to appropriate directory
3. Create PR
4. Get required approvals (minimum 2)
5. Merge PR
6. Execute via Git for SQL dashboard
